A Few Accomplishments


Miklat Finder

Miklat Finder is an application enabling users to locate the nearest bomb shelters (miklats) based on their current location or a specified address within a given city. The bomb shelter database was built by sourcing information from the city's official website and through crowdsourcing efforts from local residents.

Convert PDF scans of Rashi script books into a digital library via OCR

At Dicta, I lead the OCR pipeline using mainly Python and Google's Tesseract library, converting scans of Rashi-scripted rabbinic books into a free digital library.
